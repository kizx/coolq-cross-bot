## [coolq-cross-bot](https://github.com/kizx/coolq-cross-bot)
基于[酷Q](https://cqp.cc/)和[nonebot](https://nonebot.cqp.moe/)的QQ机器人

## 功能
1. steam游戏史低查询
    > 发送 `史低` 后根据提示再发送 `游戏id` ，或者直接发送 `史低 游戏id`  
    例：史低 203160
2. handsome时光机功能
